# 命令技能（demo-skill）

当用户输入以 `#` 开头的语句时，表示用户希望你执行某个命令。你需要识别命令名称与参数，并决定是否调用 `exe_skill_tool` 执行。

## 命令解析规则

- 以 `#` 开头，紧随其后的连续字母为命令名称。
- 命令名之后的内容（去掉首尾空格）作为参数文本。

## 支持命令

### 1) #time

- 无参数。
- 作用：返回当前系统时间。
- 输出格式：`YYYY-MM-DD HH:mm:ss`（24小时制）。

### 2) #echo

- 有一个字符串参数。
- 作用：原样复读该参数内容。
- 示例：
  - 输入：`#echo hello world`
  - 输出：`hello world`

## 调用建议

- 如果用户输入不是以 `#` 开头，不要执行该技能。
- 如果命令未知，提示可用命令：`#time`、`#echo`。
